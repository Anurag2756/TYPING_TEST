# Typing Speed Test Web App

This project is a browser-based typing speed test application designed to measure typing speed and accuracy in real time.

## Features
- Real-time typing speed calculation (WPM)
- Accuracy measurement
- Timer-based test
- Clean and responsive user interface
- Runs completely in the browser

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Live Demo
https://anurag2756.github.io/TYPING_TEST/

## Purpose
This project was developed to practice frontend development concepts and client-side logic using JavaScript.
